# Fake Review Detection using NLP and Machine Learning

## 📌 Project Overview
This project focuses on detecting fake product reviews using Natural Language Processing (NLP) and Machine Learning techniques.

## 📂 Dataset
The project uses three datasets:
- LIWC_Negative_Truthful.csv (Genuine reviews)
- LIWC_Negative_Deceptive.csv (Fake reviews)
- LIWC2015_AllReviews.csv (Combined dataset)

## ⚙️ Methodology
- Data preprocessing (cleaning, tokenization)
- Feature extraction (TF-IDF, LIWC features)
- Machine learning models:
  - Support Vector Machine (SVM)
  - Decision Tree
- Model evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1 Score

## 📊 Objective
To build a robust model that can detect fake reviews and improve trust in online platforms.

- Thesis_Project.ipynb → Main implementation
